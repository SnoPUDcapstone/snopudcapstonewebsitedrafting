Project description

For the forecasting method, BPA proposed using the persistence forecasting method due to its simplicity, which involves using the average output from previous time periods to forecast solar output for upcoming time periods. However, the proposed persistence method potentially puts significant strain on the battery, as the forecast does not take conditions other than previous PV generation into account.

Therefore, the goal of this project is to compare various forecasting methods that can improve accuracy, thereby minimizing battery utilization to reduce battery degradation. The objective of this project is to develop a model that provides accurate forecasts, measured by Mean Absolute Error and Root Mean Square Error, and limits the usage of the battery to extend the battery life.

In this project, multiple models will be analyzed for SnoPUD and BPA along with the battery degradation calculation with each model. After analysis, we will write a recommendation report on these comparisons and make a recommendation to SnoPUD most cost-effective model and present our results to BPA and SnoPUD.
